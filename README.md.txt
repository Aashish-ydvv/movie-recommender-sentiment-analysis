# 🎬 Movie Recommender + Review Sentiment Analyzer

An interactive web app built with **Streamlit**, **TensorFlow**, and the **TMDB API**.  
It recommends movies based on similarity and analyzes user reviews with a deep learning sentiment model.

## 🚀 Features
- 📌 Movie Recommendation: Get top 5 similar movies with posters, ratings, and release years.
- 📝 Review Sentiment Analysis: Fetch reviews from TMDB/IMDb and classify them as Positive or Negative.
- 🎨 Interactive UI: Built with Streamlit, featuring tabs, scrollable review containers, and color‑coded sentiment labels.
